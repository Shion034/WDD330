Reserved Words :The words cannot be used to name because these are reserved for use by the JavaScript.These words are used by the language itself.
Variables: Variables are used in programming languages to refer to a value stored in memory.Variables have to be declared before they can be used, by using the key words "const" and "let" to declare variables.
const:Used when the variable will not be reassigned to another value
let:Used if the variable minght be reassigned or changed later.
Scope: The values decleared "const"and"let" can only esists inside the block they are declared in.
1.Global Scope:Any variable declared outside of a block is said to have global scope. This means it is accessible every where in the program.
2.Local Scope: In ES6, blocks can be used to create a local scope. This means that any variables defined inside a block using the "let" or "const" will only be available inside that block and not be accessible outside of that block.
Method:A method is an action that a primitive data type or object can perform.To call a method, we use the dot operator (.) followed by the name of the method, followed by parentheses.
Template Literals:are a special types of string that were introduced in ES6. Template literals use the backtick character, `, to deliminate the string.
Symbols:were introduced as a new primitive value in ES6. They can be used to create unique values, which helps to avoid any naming collisions.
